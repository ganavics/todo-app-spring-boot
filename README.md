# Full Stack To-Do List Application

## Overview
A Full Stack To-Do List Application developed using Spring Boot, Spring MVC, Spring Data JPA, MySQL, Thymeleaf, and Bootstrap. The application allows users to efficiently manage daily tasks with CRUD operations.

## Features
- Add a new task
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Store tasks in MySQL Database
- Responsive Bootstrap UI

## Technologies Used

- Java 17
- Spring Boot
- Spring MVC
- Spring Data JPA
- MySQL
- Thymeleaf
- Bootstrap 5
- Maven
- Git & GitHub

## How to Run the Project

### Prerequisites
- Java 17
- Maven
- MySQL
- IDE (VS Code or IntelliJ)

### Steps

1. Clone the repository
```bash
git clone https://github.com/ganavics/todo-app-spring-boot.git
```
2. Open the project in your IDE.
3. Create a MySQL database named:
```
todo_db
```
4. Update the database credentials in:
```
src/main/resources/application.properties
```
5. Run the Spring Boot application.
6. Open your browser and visit:
```
http://localhost:8081
```
